## ¿Cómo se usa? 📖

El sistema cuenta con una serie de perfiles los cuales permiten hacer determinadas acciones:

Usuarios:
* **Administrador**: este usuario tiene permisos para dar de alta, visualizar, modificar y eliminar todos los usuarios y elementos dentro del sistema.
* **Investigador**: puede visualizar toda la información cargada dentro del sistema.
* **Vigilante**: puede visualizar su información personal(nombre, apellido, edad, etc) y qué sucursal lo ha contratado y para qué fecha.

Cada usuario tiene permisos para determinadas pantallas según el perfil asignado.

La versión actual del sistema permite:
* Consultar, dar de alta, modificar y eliminar en los siguientes menues:

  * Menú personas
      * Usuarios
      * Jueces
      * Delincuentes
      
  * Menú bancos
      * Entidades bancarias
      * Sucursales
      * Contratos
  
  * Menú delincunetes
      * Bandas
      * Delitos
      * Sentencias
